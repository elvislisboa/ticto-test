# How to run

# Clone Repository
$ git clone https://github.com/elvislisboa/ticto-test

# Install Dependencies
$ yarn

# Run Aplication
$ yarn start

# Run Test
$ yarn test


<p>
  Go to http://localhost:3000/ to see the result Web.
</p>
